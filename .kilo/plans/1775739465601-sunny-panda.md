# Fix: Server Action passed to Client Component Error

## Problem
The `createBooking` function is defined in a Server Component (`app/booking/page.tsx`) and passed directly to the Client Component `BookingForm`. In Next.js 16 with Turbopack, passing functions from server to client components is not allowed.

## Solution
Add `"use server"` directive to the `createBooking` function to explicitly expose it as a Server Action.

## Changes Required
1. **File:** `/workspaces/CRMEF/crmef/app/booking/page.tsx`
   - Add `"use server"` directive at the top of the `createBooking` function (line 13)

## Implementation
```typescript
// Change from:
const createBooking = async (formData: FormData) => {

// To:
const createBooking = async (formData: FormData): Promise<void> => {
  "use server";
  // rest of the function...
```

This will make the function a proper Server Action that can be called from the Client Component.
