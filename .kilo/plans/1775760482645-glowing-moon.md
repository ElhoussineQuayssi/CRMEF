# Plan: Fix React Server Action Form Method Error

## Problem
The form in `BookingForm.tsx` uses a Server Action (`createBooking`) as its action, but also specifies `method="POST"`. React automatically handles the method when using a function action, so specifying it causes the error: "Cannot specify a encType or method for a form that specifies a function as the action."

## Solution
Remove the `method` attribute from the `<form>` element in `BookingForm.tsx`.

## Steps
1. Edit `crmef/components/BookingForm.tsx` line 47 to remove `method="POST"` from the form element.
