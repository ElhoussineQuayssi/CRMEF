# Fix: React Form Action Error

## Problem
The error "Cannot specify a encType or method for a form that specifies a function as the action" occurs because:
- Line 47 of `BookingForm.tsx` uses `<form action={action} method="POST" ...>`
- When `action` is a function (Next.js Server Action), React automatically handles the form submission
- Specifying `method="POST"` conflicts with React's automatic handling

## Solution
Remove the `method="POST"` attribute from the form element in `crmef/components/BookingForm.tsx:47`

## Changes
- **File:** `crmef/components/BookingForm.tsx`
- **Line 47:** Change `<form action={action} method="POST" className="grid gap-6 text-slate-100">` to `<form action={action} className="grid gap-6 text-slate-100">`

## Verification
After the fix, the booking form should still work correctly with the Server Action (`createBooking`) passed from `crmef/app/booking/page.tsx:27`.
